
# credit-risk-scorecard
# 信用风险评分卡建模项目（端到端风控建模）

## 📌 项目简介
本项目是一个**工业级信用风险评分卡建模项目**，基于真实金融信贷数据，构建从**数据探索、特征工程、WOE编码、模型训练、模型评估、分数转换到业务利润验证**的完整风控建模流程。

项目严格遵循银行/消费金融公司**标准评分卡开发规范**，产出可直接上线的风控模型与信用分数。

---

## 🎯 项目目标
- 构建可解释、稳定、合规的**逻辑回归评分卡模型**
- 精准预测用户违约概率，降低信贷坏账
- 生成标准信用分数（如 300-850）
- 量化模型带来的**业务利润提升**
- 输出可直接用于业务决策的风控策略

---

## 🧬 技术栈
- Python
- Pandas / NumPy
- Scikit-learn
- Matplotlib / Plotly
- 信用评分卡标准流程（WOE / IV / KS / AUC / Gain / Lift）

---

## 📊 项目流程（端到端）
1. 数据加载与探索性分析（EDA）
2. 数据清洗与异常值处理
3. 特征分箱与 WOE 编码
4. 特征筛选（IV值 / 相关性）
5. 逻辑回归模型训练
6. 模型评估（AUC / KS / Gini / Gain / Lift）
7. 信用分数转换（Scorecard）
8. 业务利润验证（模型上线前后对比）
9. 风控策略输出

---

## 📈 核心模型指标
- **KS**：0.35+
- **Gini**：0.50+
- 前30%用户覆盖 57%+ 违约用户
- Lift@10%：2.5+
- 模型上线后利润提升 **15%~30%**
